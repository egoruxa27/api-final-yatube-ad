# API Yatube(Soclial Network Backend)

## Описание проекта

API Yatube это часть Backend приложения социальной сети реализованного на Django REST Framework

Основная цель проекта — практика построения REST API, работы с сериализаторами, валидаторами и связями моделей в Django.

## Основной функционал

* Аутентификация пользователей JWT
* Возможность публиковать свои посты, а также редактировать их
* просмотр постов опубликованных другими пользователями
* Подписка на других пользователей
* возможность комментировать записи, и редактировать свои комментарии
* вступать в определенные группы по интересам

## Установка и запуск проекта на локальном сервере
---
### 1. Клонировать репазиторий

```
git clone <URL_репозитория>
cd <имя_проекта>
```
---
### 2. Создать виртуальное окружение

```
python -m venv venv
```
 Windows:
```
venv\Scripts\activate
```

Mac/Linux:
```
source venv/bin/activate
```
---
### Установка зависемостей

```
pip install -r requirements.txt
```
---
### Применить миграции

```
python manage.py migrate
```
---
### Запустить сервер

```
python manage.py runserver
```
---
## Примеры API запросов

### Создание публикации

Добавление новой публикации в коллекцию.  
Доступно только для авторизованных пользователей.



#### Endpoint

**POST** `/api/v1/posts/`



#### Тело запроса
text обязательное поле
```
{
  "text": "Мой первый пост",
  "image": "base64 или ссылка на изображение",
  "group": 1
}
```


#### Response
```
{
  "id": 0,
  "author": "string",
  "text": "string",
  "pub_date": "2019-08-24T14:15:22Z",
  "image": "string",
  "group": 0
}
```
